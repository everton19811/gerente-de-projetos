# 📜 Project Charter — ONG Desenvolve

**Projeto:** Plataforma Digital Integrada
**Sponsor:** Diretoria Executiva
**Gerente de Projeto:** Everton
**Data de Início:** 01/03/2026 | **Go-Live Alvo:** 01/09/2026

## Objetivo
Substituir o ecossistema de planilhas por um sistema web integrado (CRM + Conteúdo + BI + LGPD).

## Escopo Alto Nível
✅ CRM de beneficiários, voluntários, doadores
✅ Portal editorial com workflow de aprovação
✅ Integração com gateway de doações (Pagar.me/Stripe)
✅ Dashboards executivos
✅ Módulo de consentimento e trilha LGPD
❌ App mobile nativo (fase 2)
❌ E-commerce de cursos (fase 2)

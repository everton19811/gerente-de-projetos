# ✍️ Fluxo Editorial de Artigos

## Estados
`Rascunho → Em Revisão → Aprovado → Agendado → Publicado → Arquivado`

## Regras
- Todo artigo precisa de: título, autor, categoria, resumo, capa, tags, base científica (quando aplicável)
- Revisão obrigatória por 2 pessoas (dupla checagem)
- Publicação sincroniza com o site público via API
- Histórico de versões preservado (auditoria)

## Métricas
- Tempo médio Rascunho → Publicado
- Volume publicado/mês
- Taxa de rejeição em revisão

# 🏃 Plano de Sprints (6 meses / 12 sprints de 2 sem)

| Sprint | Tema | Entregas |
|---|---|---|
| S1-S2 | Fundação | Setup infra, IAM, modelo de dados, CI/CD |
| S3-S4 | CRM Core | CRUD pessoas, famílias, interações, busca |
| S5-S6 | Portal Editorial | Workflow artigo, revisão, publicação, categorias |
| S7-S8 | Integrações | Formulários site, gateway doações, ERP contábil, e-mail mkt |
| S9 | Migração | ETL das planilhas + validação por área |
| S10 | BI | Dashboards executivo, editorial, captação |
| S11 | UAT | Testes com usuários reais, ajustes |
| S12 | Cutover | Go-live + hypercare |

# 📋 Declaração de Escopo

## Incluído
- Cadastro 360º (beneficiários, famílias, voluntários, doadores, parceiros)
- Módulo editorial com fluxo Rascunho → Revisão → Aprovação → Publicação
- Integração com formulários públicos do site atual (desenvolvenoespectro.com.br)
- Integração com gateway de doações e ERP contábil
- Dashboards em BI (Metabase/PowerBI)
- Governança LGPD: consentimento, base legal, DPIA, trilha de auditoria
- Migração de planilhas legadas (data cleaning + de-duplicação)
- Treinamento de 3 turmas de usuários

## Excluído
- App mobile nativo
- E-commerce
- Automação de marketing avançada
- Substituição do WordPress público (integração apenas)

## Premissas
- Sponsor disponível semanalmente
- Voluntários TI dedicando 8h/semana
- Infraestrutura em nuvem (AWS/Cloudflare)

## Restrições
- Budget teto: R$ 180k
- Prazo fixo: 6 meses
- LGPD obrigatória antes do go-live

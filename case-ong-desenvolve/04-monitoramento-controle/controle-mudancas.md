# 🔁 Controle Integrado de Mudanças (RFC)

## Fluxo
1. Solicitação registrada (issue GitHub / Jira)
2. Avaliação de impacto (prazo, custo, escopo, qualidade, LGPD)
3. Aprovação CCB (Change Control Board): Sponsor + GP + DPO
4. Atualização de baseline e comunicação

## Modelo RFC
- ID / Título / Solicitante / Data
- Descrição da mudança
- Justificativa de negócio
- Impacto (P/M/A) e estimativa
- Decisão: Aprovado / Rejeitado / Adiado
